# R-Adapt

Project page for **R-Adapt: What Makes VLMs Robust? Towards Reconciling Robustness and Accuracy in Vision-Language Models**.

## Local Preview

Run a simple static server from the repository root:

```bash
python3 -m http.server 8000
```

Then open `http://127.0.0.1:8000`.

## Files

- `index.html`: single-page project homepage
- `static/css/index.css`: page styling
- `static/js/index.js`: small interactions
- `static/images/`: exported figures from the paper
- `static/files/r-adapt.bib`: BibTeX entry

## Notes

- The page is designed for GitHub Pages style static hosting.
- The current repo does not include a compiled paper PDF. A PDF button can be added later by placing the final manuscript under `static/files/`.
